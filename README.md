# 🧊 SeqShape (Key Path 3D)

A lightweight, browser-native 3D Multi-Factor Authentication (MFA) demo. It replaces static passwords with a dynamic 3D spatial shape-sequencing captcha.

## 🚀 Features
* **3D Shape Captcha:** Renders 6 distinct WebGL geometries using Three.js and `OrbitControls`.
* **Dynamic Shuffling:** Shuffles layouts every session to prevent brute-forcing and muscle-memory macros.
* **Zero Frameworks:** Built entirely with vanilla JS, HTML5, and CSS custom properties. No build steps.
* **Responsive:** Mobile-first fluid design that scales from 320px to 4K.

## 🛠️ Tech Stack
* **Three.js (r128)** & WebGL for 3D rendering.
* **HTML5 Canvas 2D** for the animated background particle mesh.
* **Vanilla ES2020+ JavaScript** for the core multi-step auth logic.
