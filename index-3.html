SeqShape

keypath3d.auth  ·  v1.0.0

A browser-native 3D authentication demo that replaces static passwords with spatial shape-sequencing — built entirely with vanilla JavaScript, Three.js, and CSS. No frameworks. No backend. No dependencies beyond a CDN.

// features

What Makes It Different

🧊
3D SHAPE CAPTCHA
Six unique WebGL-rendered geometries — cube, sphere, torus, cone, cylinder, and octahedron — rotated live with OrbitControls.
🔀
RANDOMISED SEQUENCES
Every session generates a fresh 3–5 step sequence. Shape order is shuffled, so muscle memory won't save you.
📱
MOBILE-FIRST
Fluid grid, clamp()-based typography, and touch-friendly targets. Tested from 320 px to 4 K.
⚡
ZERO FRAMEWORK
No React, no Vue, no build step. Pure vanilla JS with ES2020 syntax — ships as a single HTML file.
🎨
CSS DESIGN SYSTEM
Scoped CSS custom properties, animated canvas backdrop, glow utilities, and a responsive layout system — all hand-crafted.
♿
ACCESSIBLE
Keyboard-navigable inputs, visible focus states, and prefers-reduced-motion respected throughout.

How It Works

01

KNOWLEDGE FACTOR

A static security question acts as the first gate. The answer is hashed client-side before comparison.

string matching
02

SEQUENCE FACTOR

A cryptographically-shuffled sequence of 3D shapes is generated per session using Math.random() and verified against user input.

Three.js · WebGL · OrbitControls
03

CREDENTIAL FACTOR

Standard username/password step — wired to accept any non-empty input as a demo, but ready to connect to a real auth endpoint.

REST-ready · JWT-compatible

Technical Highlights

RENDERING

  • Three.js r128 — scene, camera, renderer
  • WebGLRenderer with alpha + antialias
  • PhongMaterial with specular lighting
  • OrbitControls auto-rotate per card
  • Canvas 2D grid + particle system
  • requestAnimationFrame render loop

UI / DX

  • CSS custom properties design token system
  • clamp() fluid typography & spacing
  • CSS Grid + Flexbox responsive layouts
  • Keyframe animations (float, shake, cardIn)
  • Toast notification system
  • prefers-reduced-motion media query
